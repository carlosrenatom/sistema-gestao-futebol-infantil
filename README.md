# ⚽ Sistema de Apoio e Organização do Futebol Infantil  
### Balneário Arroio do Silva — Projeto de Extensão

Este repositório contém um sistema completo, moderno e funcional desenvolvido para apoiar a gestão do **programa municipal de futebol infantil** em Balneário Arroio do Silva.  
O projeto foi criado como parte de uma atividade extensionista, com foco em tecnologia aplicada ao esporte, organização de treinos, presença e comunicação com responsáveis.

---

## 🚀 Funcionalidades

### 👦 Cadastro de Atletas
- Nome, idade, turma e responsável  
- Edição e exclusão  
- Armazenamento em *localStorage*  

### 📅 Gestão de Treinos
- Agendamento com data, hora e turma  
- Lista de treinos futuros  
- Atualização automática no painel (dashboard)

### ✔️ Registro de Presença
- Seleção por turma  
- Marcação rápida “presente/ausente”  
- Histórico salvo no navegador  
- Cálculo automático de presença

### 📢 Comunicados
- Envio de comunicados internos  
- Histórico dos últimos recados  
- Registro de data e hora

### 📊 Dashboard Inteligente
- Total de atletas  
- Total de treinos  
- Média de presença  
- Gráfico por turma (Chart.js)  

### 📤 Exportação e Relatórios
- Exportação CSV dos atletas  
- Geração de relatório .txt  
- Estatísticas gerais automáticas  

---

## 🧠 Tecnologias Utilizadas

- **HTML5**
- **CSS3 (layout moderno e responsivo)**
- **JavaScript (Vanilla)**
- **Chart.js** para gráficos
- **LocalStorage** para persistência
- **Design baseado em UI moderna (Dashboard)**
